# Temporal Part and Boundaries（时间组成与边界）

> 📄 **完整分析** → [[../详细分析/DM2-TemporalPartAndBoundaries详细分析]]

## 一句话
DM2 的时间建模系统——**时间段可以重叠！** 边界是一等实体。

## 双轨建模

```
轨道 A: temporalWholePart（时间组成）
  ┌─────────┐     ┌─────────┐
  │  Q1  │←wp─│  Q2  │       大时间段由小时间段组成
  └────┬────┘     └────┬────┘
       │               │
  ┌────┴────┐     ┌────┴────┐
  │Jan│Feb│Mar│     │Apr│May│Jun│
  └─────────┘     └─────────┘

轨道 B: temporalBoundary（时间边界）
  ┌─────────────────┐
  │     时间段       │
  ├── startBoundary ──→ StartBoundaryType (可度量！)
  └── endBoundary ────→ EndBoundaryType (可度量！)
```

## ⭐ 关键特性：时间可重叠！

不同于空间 WholePart（一只手不能属于两个人），**时间段可以重叠**：

```
项目A:  ██████████  (1月~3月)
项目B:      ████████████  (2月~5月)
         ↑↑ 2月是两个项目的重叠部分 — 这是合法的！
```

## 双层边界设计

| 层级 | 内容 | 示例 |
|------|------|------|
| **Instance 层** | 具体边界值 | "2026-01-01 00:00:00" |
| **Type 层** | 统计约束 | "项目开始时间必须在 Q1 内" |

Measure 可以无缝集成到边界上——使时间变成**可计算的**。

## 三种时间建模选择

| 模式 | 适用场景 | 关联类型 |
|------|---------|---------|
| **temporalWholePart** | 阶段/季度/层级时间 | 时间组成 |
| **BeforeAfter** | 纯时序（先后关系） | 因果/前置 |
| **temporalExtent** | 只关心起止点 | 最简单 |
