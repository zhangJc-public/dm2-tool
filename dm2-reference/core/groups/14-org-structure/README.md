# Organizational Structure（组织结构关系）

> 📄 **完整分析** → [[../详细分析/DM2-OrgStructure详细分析]]

本目录关注**组织之间的关系**（非组织内部层级）。

## 核心区分

| 关系类型 | 模式 | 示例 | 刚性 |
|---------|------|------|------|
| **内部层级** | WholePart | 华北区 → 北京办 → 安全部 | 🔴 强（互斥） |
| **跨组织协作** | Overlap | 张三同时属于项目组A和职能部门B | 🟡 弱（可交叉） |

## Overlap 模型的典型应用

```
          项目组A ───Overlap─── 张三 ───Overlap─── 职能部门B
               ↑                                      ↑
          (项目角色)                               (职能角色)
```

## OV-4 组织视图的关系基础

OV-4 不仅画组织树，更要画组织间的 **Overlap 关系网**——这是 DM2 区别于普通 OrgChart 的关键差异。
