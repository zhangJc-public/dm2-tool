# 16-InformationAndData

## 📌 一句话说明
**信息与数据——Sign/Rep/Data 三层表示体系。** Peirce 符号学三元模型在 DM2 中的形式化。

## 🎯 目录用途
- Sign（符号/标记）：可感知的模式（如：文字、图标、波形）
- Representation（表示形式）：数据的编码方式（如：PDF、JSON、数据库行）
- Data（数据内容）：语义化的信息
- 三层的 representedBy 链：Sign ← representedBy ← Representation ← representedBy ← Data
