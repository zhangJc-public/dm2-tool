# OV-4: Organizational Relationships Chart（组织关系图）

## 一、定位与目的

### 1.1 这张图回答什么问题？

**一句话**：**谁**参与这个架构？组织之间是什么关系？

OV-4 描述架构中的**组织结构和交互关系**——回答"人"和组织维度的关键问题。

核心问题集：
| 问题 | 回答 |
|------|------|
| 涉及哪些组织和角色？ | 军事/民事组织的完整列表 |
| 组织间有什么层级关系？ | 指挥结构、组成关系 |
| 组织间如何协作？ | C2 关系、协作、监督报告等交互 |
| 谁是关键干系人？ | 架构的参与者识别 |

## 二、核心内容解析

### 2.1 必要性说明

> OV-4 展示**组织结构和交互**。所示的组织可以是军事或民事。
>
> **OV-4 以两种形式存在**：
> 1. **角色型（Role-based）** —— 如典型的旅指挥结构
> 2. **实际型（Actual）** —— 如某部门或机构的组织架构图

#### 角色型 OV-4

展示**组织资源之间的可能关系**：
- **关键关系类型**：**组成（composition）**——即一个组织资源是父组织的一部分
- 可展示每个组织资源的**角色（Roles）**
- 展示角色之间的**交互**——即角色代表组织资源的**功能方面**

⚠️ DoDAF V2.0 **未规定固定的资源交互类型**：
- 架构师应从 DM2 中选择适当的交互类型，或添加新的

交互示例：监督报告、**指挥与控制（C2）**关系、协作等

#### 实际型 OV-4

展示**特定时间点的真实组织结构**：
- 用于为架构的其他部分（如 AV-1 和 CV 系列）提供上下文

### 2.2 推荐表达形式

| 形式 | 适用场景 | 备注 |
|------|---------|------|
| **组织结构图** | ⭐ 默认形式 | 层级+交互关系 |
| 矩阵（RACI） | 职责分配 | 角色与活动的交叉 |
| 混合图表 | 复杂场景 | 同时展示典型和实际结构 |

### 2.3 关键数据元素

| 元素 | 说明 |
|------|------|
| **Organizational Resource** | 组织资源（组织/角色） |
| **Composition Relation** | 组成关系（整体-部分） |
| **Interaction Relation** | 交互关系（C2、协作、监督等） |
| **Role** | 组织资源承担的角色 |
| **Billet / Person Type** | 职位或人员类型（实际型中可详细描述） |

⚠️ **重要约束**：DoDAF 中不查看**个人（individual people）**，但具体的**职位（billets）** 或**人员类型（Person Types）** 可以在实际型 OV-4 中详细描述。

### 2.4 完整细节

#### 2.4.1 组织关系的两大类别

官方定义两类基本关系：

| 类别 | 子类型 | 说明 |
|------|--------|------|
| **结构关系 (Structure)** | Whole-Part（组成） | 一个组织是另一个的一部分 |
| **交互关系 (Interaction)** | 多种子类型 | 指挥控制、协作、客户-供应商等 |

#### 2.4.2 非层级交互关系

> 在典型和特定情况下，都可以叠加**资源交互关系**——表示组织元素之间的**非严格层级**关系（如**客户-供应商关系**）。

#### 2.4.3 OV-4 与其他视图的关系

| OV-4 出现在其他视图中的方式 | 说明 |
|--------------------------|------|
| SV-1 中作为能力的**组织组成部分** | 角色型 OV-4 的组织资源可作为能力或资源的组成 |
| PV-1 中作为项目的**拥有组织** | 实际型组织拥有项目 |
| SV-1 中作为**部署能力的要素** | 实际组织形成在系统层面实现需求的已部署能力 |

#### 2.4.4 组织关系的重要性

> 组织关系在架构模型中很重要，因为它们可以阐明：
> 1. **基本人力角色**（如需要什么技能来执行作战活动）
> 2. **管理关系**（如指挥结构或与其他关键参与者的关系）
>
> 此外，组织关系是使用 Needlines 查看的某些**协作需求的驱动因素**。

## 五、术语表

| 英文术语 | 中文翻译 | 说明 |
|---------|---------|------|
| Role-based OV-4 | 角色型 OV-4 | 展示典型/通用组织结构的抽象形式 |
| Actual OV-4 | 实际型 OV-4 | 展示真实组织在特定时刻的结构 |
| Billet | 职位 | 组织中的一个正式岗位（非具体个人） |
| Person Type | 人员类型 | 某类人员的分类（如技能等级） |
| Command and Control (C2) | 指挥与控制 | 军事组织中的指挥链关系 |
| Composition | 组成关系 | 整体-部分的结构关系 |
