# PV-3: Project to Capability Mapping（项目到能力映射）

## 一、定位与目的

### 1.1 这张表回答什么问题？

**一句话**：哪个项目交付了哪项能力？有没有遗漏或重复？

PV-3 是**项目和能力的双向追溯矩阵**——确保每项能力都有项目负责，每个项目都有明确的能力产出。

核心问题集：
| 问题 | 回答 |
|------|------|
| 能力 A 由哪个项目交付？ | 能力→项目追溯 |
| 项目 B 支持了哪些能力？ | 项目→能力覆盖 |
| 有没有无人认领的能力？ | 缺口识别 |
| 是否有多个项目做同一件事？ | 冗余检测 |

## 二、核心内容解析

### 2.1 必要性说明

> PV-3 描述**项目与能力之间的映射关系**。

它是 PV-1/PV-2 和 CV 系列之间的**一致性校验工具**。

### 2.2 与 PV-1/PV-2 的区别

| 维度 | PV-1 | PV-2 | PV-3 |
|------|------|------|------|
| **焦点** | 项目是什么 | 什么时候做 | 做→什么的对应 |
| **形式** | 清单/分组 | 时间线 | 矩阵 |
| **价值** | 总览 | 进度规划 | 追溯/审计 |
