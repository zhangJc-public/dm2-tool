# 厂商

## 角色定义

**厂商**是提供安全产品的供应商，包括硬件设备、软件许可、云服务等。

## DM2 类型层级

- **Organization-Type**：`厂商`
- **关联**：performerTypeOfIndividual → Organization-Individual（具体厂商）
- **关联**：performerTypeOfIndividual → System-Individual（厂商产品）

## 职责范围

| 职责 | 说明 |
|------|------|
| 产品提供 | 交付符合规格的产品 |
| 售前支持 | 技术咨询、方案配合 |
| 售后服务 | 质保、维修、升级 |
| 培训支持 | 产品培训、技术认证 |

## 典型关切点

- ✅ 产品性能是否达标
- ✅ 技术支持是否及时
- ✅ 资质认证是否齐全
- ✅ 品牌口碑是否良好

## 关联角色

| 角色 | 关系 |
|------|------|
| 集成商 | 产品供货 |
| 甲方 | 最终用户 |
| 乙方 | 合作关系 |
| 建设方 | 技术支持 |
