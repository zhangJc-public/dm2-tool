# 集成商

## 角色定义

**集成商**是负责多厂商设备/软件集成、接口调试、系统联调的组织。

## DM2 类型层级

- **Organization-Type**：`集成商`
- **关联**：performerTypeOfIndividual → Organization-Individual（具体集成商）

## 职责范围

| 职责 | 说明 |
|------|------|
| 设备集成 | 多厂商设备互联互通 |
| 软件集成 | 中间件、平台软件集成 |
| 接口调试 | API、协议对接 |
| 系统联调 | 全系统功能验证 |

## 典型关切点

- ✅ 设备接口是否兼容
- ✅ 集成风险是否可控
- ✅ 调试周期是否充足
- ✅ 技术支持是否到位

## 关联角色

| 角色 | 关系 |
|------|------|
| 甲方 | 服务对象 |
| 厂商 | 设备/软件来源 |
| 建设方 | 协同实施 |
| 设计方 | 集成方案依据 |
