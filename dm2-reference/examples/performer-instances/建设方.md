# 建设方

## 角色定义

**建设方**是负责系统部署、设备安装、系统调试的施工单位。

## DM2 类型层级

- **Organization-Type**：`建设方`
- **关联**：performerTypeOfIndividual → Organization-Individual（具体施工单位）

## 职责范围

| 职责 | 说明 |
|------|------|
| 设备安装 | 硬件设备上架、接线 |
| 系统部署 | 软件安装、网络配置 |
| 系统调试 | 功能测试、性能调优 |
| 试运行 | 系统试运行、问题整改 |

## 典型关切点

- ✅ 施工环境是否符合要求
- ✅ 工期是否合理
- ✅ 技术规范是否明确
- ✅ 验收标准是否清晰

## 关联角色

| 角色 | 关系 |
|------|------|
| 甲方 | 服务对象 |
| 设计方 | 设计交底 |
| 监理方 | 接受监督 |
| 乙方 | 协同实施 |
