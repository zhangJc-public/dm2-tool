# 最终用户

## 角色定义

**最终用户**是使用安全系统进行日常业务操作的终端人员所属组织。

## DM2 类型层级

- **Organization-Type**：`最终用户`
- **关联**：performerTypeOfIndividual → Organization-Individual（具体用户组织）

## 职责范围

| 职责 | 说明 |
|------|------|
| 系统使用 | 按规范使用系统功能 |
| 日常巡检 | 例行安全检查 |
| 事件报告 | 发现问题及时报告 |
| 培训参与 | 参加安全培训 |

## 典型关切点

- ✅ 操作是否简便
- ✅ 响应是否及时
- ✅ 功能是否实用
- ✅ 培训是否充分

## 关联角色

| 角色 | 关系 |
|------|------|
| 甲方 | 归属单位 |
| 建设方 | 使用培训对象 |
| 监管方 | 合规执行者 |
