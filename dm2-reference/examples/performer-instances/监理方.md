# 监理方

## 角色定义

**监理方**是受甲方委托，负责质量把控、进度监督、风险预警的独立第三方。

## DM2 类型层级

- **Organization-Type**：`监理方`
- **关联**：performerTypeOfIndividual → Organization-Individual（具体监理单位）

## 职责范围

| 职责 | 说明 |
|------|------|
| 质量监督 | 检查施工质量、设备规格 |
| 进度监控 | 跟踪项目里程碑 |
| 风险预警 | 识别并报告项目风险 |
| 文档审核 | 审核技术文档、验收材料 |

## 典型关切点

- ✅ 质量标准是否达标
- ✅ 进度是否按计划
- ✅ 风险是否可控
- ✅ 文档是否合规

## 关联角色

| 角色 | 关系 |
|------|------|
| 甲方 | 委托监督 |
| 建设方 | 监督实施 |
| 设计方 | 监督设计 |
| 乙方 | 监督履约 |
