# 监管方

## 角色定义

**监管方**是负责合规检查、等级保护、资质审批的监管或合规机构。

## DM2 类型层级

- **Organization-Type**：`监管方`
- **关联**：performerTypeOfIndividual → Organization-Individual（具体监管机构）

## 职责范围

| 职责 | 说明 |
|------|------|
| 标准制定 | 发布合规标准、规范 |
| 资质审批 | 颁发资质证书 |
| 合规检查 | 定期检查、专项检查 |
| 处罚处理 | 违规处理、整改要求 |

## 典型关切点

- ✅ 是否符合等级保护要求
- ✅ 资质证书是否有效
- ✅ 安全措施是否到位
- ✅ 持续合规状态

## 关联角色

| 角色 | 关系 |
|------|------|
| 甲方 | 监管对象 |
| 乙方 | 连带监管 |
| 设计方 | 合规设计要求 |
| 建设方 | 建设合规要求 |
