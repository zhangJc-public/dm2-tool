# 乙方（承包商/服务方）

## 角色定义

**乙方**是接受甲方委托，负责履约、提供产品或服务的组织或个人。

## DM2 类型层级

- **Organization-Type**：`乙方`
- **关联**：performerTypeOfIndividual → Organization-Individual（具体乙方组织）

## 职责范围

| 职责 | 说明 |
|------|------|
| 合同履约 | 按约提供产品/服务 |
| 交付质量 | 确保交付物符合规格 |
| 技术支持 | 提供售后服务 |
| 风险承担 | 承担实施风险 |

## 典型关切点

- ✅ 合同条款是否清晰
- ✅ 付款条件是否合理
- ✅ 技术方案是否可行
- ✅ 责任边界是否明确

## 关联角色

| 角色 | 关系 |
|------|------|
| 甲方 | 服务对象 |
| 厂商 | 产品来源 |
| 集成商 | 协同交付 |
| 设计方 | 协同实施 |
| 建设方 | 协同实施 |
