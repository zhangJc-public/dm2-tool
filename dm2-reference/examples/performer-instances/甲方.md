# 甲方（采购方/投资方）

## 角色定义

**甲方**是提出安全体系建设需求、提供资金支持、并对交付成果进行验收的组织或个人。

## DM2 类型层级

- **Organization-Type**：`甲方`
- **关联**：performerTypeOfIndividual → Organization-Individual（具体甲方组织）

## 职责范围

| 职责 | 说明 |
|------|------|
| 需求提出 | 明确业务目标、安全合规要求 |
| 预算控制 | 审批投资、监控成本 |
| 进度监督 | 阶段性评审、里程碑验收 |
| 成果验收 | 最终交付物评审与确认 |

## 典型关切点

- ✅ 需求是否被完整实现
- ✅ 项目是否按期交付
- ✅ 投资回报率（ROI）
- ✅ 合规性是否满足

## 关联角色

| 角色 | 关系 |
|------|------|
| 设计方 | 委托设计 |
| 建设方 | 委托实施 |
| 监理方 | 委托监理 |
| 监管方 | 接受监管 |
| 乙方 | 委托采购 |
