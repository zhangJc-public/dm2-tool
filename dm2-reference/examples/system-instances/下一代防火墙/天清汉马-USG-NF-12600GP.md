# 天清汉马 USG-NF-12600GP

> [!info] 产品型号
> - **型号**：USG-NF-12600GP
> - **系列**：GP 系列
> - **定位**：万兆级数据中心防火墙
> - **CPU**：4核，支持超线程（显示8核）
> - **典型吞吐量**：万兆级

## 基本规格

| 项目 | 规格 |
|:----:|------|
| CPU物理核数 | 4核，支持超线程 |
| 界面显示核数 | 8核 |
| 默认网络接口 | 1xGE（用于型号定制，不单独销售） |
| 标配扩展槽 | 3个（可定制扩展至6个） |
| 最大千兆接口(标准) | 25个 |
| 最大千兆接口(增强) | 49个（需搭配槽位扩展模块） |
| 最大万兆接口 | 16个（需搭配槽位扩展模块） |

## 可选扩展模块

| 模块型号 | 说明 |
|---------|------|
| USG-NF-12600GP-4GE | 4GE扩展卡 |
| USG-NF-12600GP-4GEBP | 4GE + 2对Bypass扩展卡 |
| USG-NF-12600GP-4SFP | 4SFP扩展卡 |
| USG-NF-12600GP-8GE | 8GE扩展卡 |
| USG-NF-12600GP-8GEBP | 8GE + 2对Bypass扩展卡 |
| USG-NF-12600GP-8SFP | 8SFP扩展卡 |
| USG-NF-12600GP-2SFI | 2万兆SFP+扩展卡 |

## 可选功能模块

| 模块型号 | 说明 |
|---------|------|
| USG-NF-12600GP-IPS-1/2 | IPS特征库1年/2年升级 |
| USG-NF-12600GP-AV | AV模块开通 |
| USG-NF-12600GP-AV-1/2 | 病毒库1年/2年升级 |
| USG-NF-12600GP-AVE | 增强型AV模块 |
| USG-NF-12600GP-AC-1/2 | 上网行为1年/2年升级 |
| USG-NF-12600GP-NF-1Y/2Y | NF特征打包升级（IPS+上网行为） |

## 适用场景

- 数据中心万兆边界
- 大型业务系统防护
- 多业务隔离与访问控制
- 万兆主干链路防护

## 注意事项

- 槽位扩展模块为受控销售
- 使用槽位扩展模块后，同一层扩展卡8电、8光、4电、2万兆和4光、4万兆互斥
- 所有SFP和SFP+网口标配不含光纤模块
