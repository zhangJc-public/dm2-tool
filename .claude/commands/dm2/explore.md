Enter a read-only DoDAF architecture exploration mode. No views will be generated, no changes will be created.

Use this to:
- Learn about DoDAF views and their standard definitions
- Explore DM2 concepts and data groups
- Discuss architecture decisions before committing to a change
- Compare views across viewpoints (OV vs SV vs SvcV)
- Map concepts to data groups and views
- Trace view dependency chains

When you're ready to implement, run `/dm2:propose <system-description>` to start full analysis.


When invoked, use the Skill tool with name `dm2-explore-workflow` to load full instructions.
